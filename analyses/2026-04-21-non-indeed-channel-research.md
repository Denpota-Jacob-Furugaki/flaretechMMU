# COBOL 人材獲得のための Indeed 以外チャネル — 事前リサーチ

**日付:** 2026-04-21
**関連セクション:** market
**背景:** 2026-04-21 の阿久根さん・野本さんとの打ち合わせで、「COBOL の言語市場と SES 業界の年齢制約の構造的ミスマッチ」を確認。**Indeed の原稿改善だけでは根本解決しない** という結論に至り、Indeed 以外のチャネル（スカウト型媒体 / 転職エージェント）への拡張を検討中。HERP 生データ到着前の **事前リサーチ** として、主要 7 サービスの料金・機能・業界フィット感を調査。

---

## 問い

COBOL 経験者（特に **金融系 SE で 20〜40代**）を確保するために、Indeed 以外にどのチャネルが筋が良いか。料金効率・リーチする人材層・Flaretech の既存スタックとの親和性の 3 観点で比較する。

---

## 方法

- 各サービスの **料金体系**（基本料 + 成功報酬率 + 返金規定）を公開情報から調査
- **得意業界・職種**（特に金融 / IT / ハイクラス）と **会員層**（年齢帯・経験値）
- Flaretech が既に使っているチャネル（Indeed / Wantedly / doda media / レバテック media / type / エン転職 等）は **除外** し、未利用のチャネルに絞る
- COBOL 求人の実在感を確認（doda / リクエージェント / マイナビ転職 / Green に COBOL 求人多数あることを確認）

### 重要な気づき

1. **media 系（既存）の方が直接コストは圧倒的に安い**。doda ¥197K は agent 系 ¥2.1M の **1/10 以下**
2. **ただし media 系は「マッチする候補者がそもそも母集団にいないと」成立しない**。COBOL のように市場供給が薄い職種では、母集団の厚いエージェント（特に金融特化）が優位
3. **リクルートダイレクトスカウト ¥900K は agent より 2.3 倍安い**。COBOL に特化していないが、潜在候補者プールは大きい
4. **BizReach は 6ヶ月でまとめ採用できると最安水準（¥1.28M/hire）**。採用数を稼げればコスパは良い
5. **エージェント型は「保険」として強い**。¥2.1M は高いが、**1 承諾=1 採用が保証される** (agentが候補者をプレフィルタ)

## Flaretech 文脈での推奨ランキング — 筋の良さ順

### 🥇 1. コトラ (KOTORA) — **COBOL x 金融 SE 最適**

- 金融・FinTech 特化で **非公開求人 10,000+件・金融系 800 社+実績**
- 案件難易度ベース料率調整あり → COBOL みたいなニッチ案件でも相談可能
- 金融系 SE の候補者データベースの厚さで他社を圧倒
- **推奨用途:** 今すぐ試すべき最優先。エン転職(承諾 0)の予算 ¥1.23M を振り替え

### 🥈 2. BizReach — **スカウト型の王道**

- 210万+ 会員のデータベース、30〜50代の潜在転職層に強い
- **金融系・ハイクラス** セクションが厚い
- 基本料 ¥85万/6ヶ月 + 成功報酬 15-25% の二段構造 → 採用ペースによっては最安水準に
- **推奨用途:** 6ヶ月コミットできるなら ROI 最高。並行で Forkwell / Wantedly も動かす

### 🥉 3. リクルートダイレクトスカウト — **低リスクで試すべき**

- **基本料ゼロ** + 成功報酬 15% の**業界最安**
- スカウト送信無制限、会員規模も大
- COBOL / 金融系に特化してはいないが、潜在候補者のプールは最大級
- 返金規定も寛容（6ヶ月以内自己都合退職で 10〜80%）
- **推奨用途:** BizReach と並行する「セカンドスカウト」として即時開始。実質無料で母集団チェック可能

### 4. JAC Recruitment — **金融特化 agent の定番**

- 35% 料率はやや高いが、外資・金融・ミドル層の候補者の質は高い
- 返金規定: 3ヶ月以内自己都合退職で 50% 返金
- **推奨用途:** コトラと並行して「もう 1 社」枠。重要職種の保険として

### 5. paiza 転職 — **若手 Java エンジニア × 移行志向に向いた二次チャネル**

- プログラミングテストで **客観的スキル評価** 済みの候補者
- COBOL テストは無いが、**Java/AWS 側からの母集団形成** に効く
- 25〜35% の **スキルランク別料率** (B ランクで 30%)
- **推奨用途:** Indeed リライトの「Java/AWS 側から移行エンジニアを引き込む」戦略と整合。長期的にスキル可視化資産になる

### 6. ギークリー — **優先度低**

- IT / Web / ゲーム特化だが、**Web 側への偏り** あり
- COBOL / 金融系 SE の候補者層は薄い可能性高
- 平均内定日数 25日は魅力だが、マッチしない層に速くリーチしても意味がない

### ⛔ レバテックキャリア — **既存と重複**

- Flaretech は既に **レバテック media** を利用中 (2026 YTD 応募 227、書類通過率 87%)
- エージェント側に切り替えても同じ候補者プールに被る可能性が高い
- 優先度は低く、他を試してから検討

## HERP データ到着後に足す分析

HERP の全チャネル生データが入手できたら、以下を追加:

1. **agent 経由の過去実績** — リクエージェント / マイナビエージェント等を過去使っていた期間の成果（承諾数・期間・コスト）
2. **年代別承諾率の比較** — Indeed の 50代+ 落ち率 vs agent 経由の承諾者の年代分布
3. **職種別承諾率** — COBOL 案件だけに絞ったチャネル別ファネル
4. **コスト・オブ・ノン・ヒア** — 空席が続いた場合の営業機会損失試算

これらが揃うと、**「エン転職 の ¥1.23M/16週 をコトラに振り替えた場合の期待 ROI」** が数字で語れるようになる。

## 留意点

- **料率・会員数は 2026-04 時点の公開情報**。契約時点での最新料率は直接見積もり必須
- **BizReach の「6ヶ月で X 承諾」計算は楽観シナリオ**。COBOL のようなニッチ職種は想定より時間がかかる可能性
- **agent の「保証あり」は候補者面接段階までの保証**。候補者辞退は返金対象外
- **外資・英語不問** の COBOL 求人は、JAC のようなグローバル特化 agent には不利な側面あり
- **既存 agent の「別会社運用」(doda / type / エン転職)** が Flaretech の 64% の応募を生んでいるため、既存関係を壊さない形での追加が望ましい
