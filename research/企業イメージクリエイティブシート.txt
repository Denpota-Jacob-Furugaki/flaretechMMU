企業イメージ クリエイティブシート
==================================================================================================

| カテゴリ            | 概要                                          | コンテンツ／クリエイティブイメージ                 | 重要度 | 優先度 |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 企業理念            | 会社のミッション・ビジョン・バリューを伝え、    | 代表メッセージ動画、理念を体現する社員の声、        |   ●    |   ●    |
|                     | 共感を得る。                                   | ブランドムービー。                                  |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 事業概要            | 主要事業・サービスを分かりやすく紹介し、        | サービス紹介ページ、事業説明インフォグラフィック、  |   ●    |   ●    |
|                     | ビジネスの全体像を理解してもらう。              | 顧客事例。                                          |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 社員・社員紹介      | 実際に働いている社員の人柄や価値観を伝え、      | 社員インタビュー、1日密着動画、                     |   ●    |   ●    |
|                     | 働くイメージを具体化する。                      | チーム紹介コンテンツ。                              |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| ★ キャリアパス・    | 入社後のキャリアステップ、研修制度、            | キャリアマップ図、研修・育成制度紹介、              |   ●    |   ●    |
|   成長機会 ★        | 成長支援の仕組みを提示する。                    | OJT事例、昇進ストーリー。                           |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 勤務環境            | オフィス・福利厚生・働き方                      | オフィスツアー動画、福利厚生一覧、                  |   ●    |   ○    |
|                     | (リモート、フレックス等)を紹介。                | 制度利用者の声。                                    |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 業界動向            | 市場トレンド・業界ポジションを共有し、          | マーケットレポート、業界解説記事、                  |   ○    |   ○    |
|                     | 将来性を示す。                                  | 経営層による業界展望コラム。                        |        |        |
|---------------------|-----------------------------------------------|----------------------------------------------------|--------|--------|
| 企業の強み          | 他社にない独自の強み・競争優位性を              | 技術・ノウハウ紹介、受賞歴、                        |   ●    |   ●    |
|                     | 具体的に提示する。                              | ケーススタディ、比較表。                            |        |        |
==================================================================================================

凡例:
  ● = 高い  /  ○ = 低い
  ★ で囲まれた行 = 重点項目 (画像の赤枠ハイライト相当)
